# Simulation Parameters

## Wheel Inertia

- Radius (r): 60mm = 0.06m
- Length (l): 40mm = 0.04m
- Mass (m): 0.5kg per wheel.

$$I_{xx} = I_{yy} = \frac{1}{12}ml^2 + \frac{1}{4}mr^2$$

$$I_{xx} = I_{yy} = \frac{1}{12}(0.5)(0.04)^2 + \frac{1}{4}(0.5)(0.06)^2 = 0.000516$$

$$I_{zz} = \frac{1}{2}mr^2$$

$$I_{zz} = \frac{1}{2}(0.5)(0.06)^2 = 0.0009$$
